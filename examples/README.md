# Wealthica Python SDK Examples

This directory contains example implementations demonstrating how to use the Wealthica Python SDK.

## Available Examples

### [basic/](./basic/)

Basic examples including:
- **`basic.py`** - Command-line script demonstrating core SDK functionality
- **`flask_server.py`** - Flask web server with REST API endpoints

See [basic/README.md](./basic/README.md) for setup instructions.

## Getting Started

1. Install the Wealthica SDK:

```bash
pip install wealthica
```

2. Get your API credentials from [Wealthica](https://wealthica.com)

3. Navigate to an example directory and follow the README instructions.
